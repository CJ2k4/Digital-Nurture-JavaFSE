# Digital Nurture 5.0 — Java Full Stack Engineer

My hands-on solutions for the Cognizant Digital Nurture 5.0 (Java FSE) Deepskilling
programme, organised by week.

## Weeks

| Week | Topics |
| --- | --- |
| [Week 1](Week1) | Design Patterns, Data Structures & Algorithms, PL/SQL, JUnit 5 & Mockito, SLF4J |
| [Week 2](Week2) | Spring Core & Maven, Spring Data JPA with Hibernate |
| [Week 3](Week3) | Spring REST with Spring Boot 3, JWT authentication |
| [Week 4](Week4) | Microservices with Spring Boot 3 and Spring Cloud |
| [Week 5](Week5) | React |
| [Week 6](Week6) | Git |

Each week has its own README with the exercises and how to run them.

## Stack

Java 17 · Spring Boot 3.2.5 · Spring Cloud 2023.0.1 · Maven 3.9 · React 18 · H2 / MySQL / Oracle

## Quick start

```bash
# Java projects
cd Week1/JUnit-Mockito-SL4J && mvn test
cd Week3/spring-learn && mvn spring-boot:run

# React
cd Week5/React/01-myfirstreact && npm install && npm start
```
